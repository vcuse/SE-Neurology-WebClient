Introduction
---------------------------------
This is the Project Page for the Neurology WebClient! This is the code to a video chat site developed by the VCUSE Lab. This site aims to video chat between multiple devices ranging from other PC's, to iPhones, to Virtual Reality Headsets.

About This Project
--------------------
This site uses WebRTC along with authentication against a cloud based server in order to serve video streams.
